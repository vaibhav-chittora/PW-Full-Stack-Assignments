
In JavaScript, there are five primitive data types:


Number:Represents numeric data, including integers, floating-point numbers, and exponential notation.

Example:
let num1 = 10;        // Integer
let num2 = 3.14;      // Floating-point number
let num3 = 2.5e6;     // Exponential notation (2.5 * 10^6)

String:Represents textual data enclosed within single or double quotes.

Example:
let name = 'John';    // Single quotes
let message = "Hello, World!";  // Double quotes

Boolean:Represents a logical value indicating true or false.

Example:
let isLogged = true;   // User is logged in
let isActive = false;  // Account is inactive

Undefined:Represents a variable that has been declared but not assigned a value.

Example:
let x;   // Variable declared but not assigned (value is undefined)

Null:Represents a deliberate absence of a value.

Example:
let data = null;   // No data available
